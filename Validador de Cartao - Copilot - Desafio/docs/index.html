Demo — Validador de Bandeira de Cartão

Este é um demo simples do script. Use a linha de comando ou veja o GIF animado abaixo.

Exemplo de uso (CLI)

$ python identificador_bandeira_cartao.py 4111111111111111 5555555555554444
4111111111111111 -> Visa | Luhn: válido
5555555555554444 -> MasterCard | Luhn: válido

Sobre

Projeto educacional para identificar bandeiras e validar Luhn. Código disponível no repositório GitHub.
