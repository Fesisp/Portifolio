Obrigado pelo interesse em contribuir!

Regras rápidas:

1. Abra uma issue descrevendo a mudança proposta antes de PRs grandes.
2. Siga o estilo PEP8 para Python. Recomendamos rodar `flake8` antes de enviar.
3. Inclua testes para novas funcionalidades ou correções.
4. Pull requests devem ser abertos contra a branch `main` e conter descrição clara do objetivo.

Configuração local rápida:

```powershell
& '.\.venv\Scripts\python.exe' -m pip install -r requisitos
& '.\.venv\Scripts\python.exe' -m pytest -q
```
